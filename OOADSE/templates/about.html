Face emotion recognition

Facial emotion recognition model has been built using the FER13 dataset. An initial facial emotion recognition model has been built using Convolutional Neural Network giving an accuracy of 66% for the test split.

Music Generation

Two music generation models were built, one with character embedding and another with word embedding. Evident from the results, the word-embedded model generated semantically meaningful sentences with an accuracy of 90%
